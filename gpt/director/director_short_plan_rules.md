# Director Short Plan Rules

Если между календарными точками есть пустое время, можно создать короткий внутренний план.

Пример:
- точка А: заселение;
- точка Б: завтрашняя проверка энергии.

Движок может решить:
- дать бытовую сцену с последствиями;
- показать Ливию не фоном;
- добавить один крючок: сообщение, звонок, слух, короткая встреча, наблюдение;
- не раскрывать hidden lore;
- не затягивать, если игрок хочет пропустить время.

## Правила

- short_plan содержит 1–3 пункта;
- создаётся только при пустом промежутке между важными событиями;
- после выполнения пункт удаляется или помечается done;
- если игрок явно хочет пропустить, можно дать один короткий meaningful hook или честно пропустить;
- не превращать short_plan в рельсы;
- не спорить с календарём.
