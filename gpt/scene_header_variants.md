# Варианты шапки сцены

## Вариант 1 — компактный технический

```txt
╭─ Академия Астрейн · 15 августа 1198
│ Время: утро / день / вечер
│ Локация: ...
│ Рядом: Ливия, ...
│ Состояние Акиры: ...
│ Сцена: ...
╰─ Ход игрока:
```

## Вариант 2 — атмосферный, но не громоздкий

```txt
Академия Астрейн · 15 августа 1198
────────────────────────────
Время: ...
Локация: ...
Рядом: ...
Акира: ...
Фокус сцены: ...
```

## Вариант 3 — карточка сцены

```txt
[Академия Астрейн]
Дата: 15 августа 1198
Время: ...
Место: ...
Активные персонажи: ...
Состояние Акиры: ...
Открытые крючки: ...
```

## Вариант 4 — самый минимальный

```txt
15 августа 1198 · Академия Астрейн
Место: ...
Рядом: ...
Состояние: ...
```

## Вариант 5 — визуальная новелла

```txt
◆ Академия Астрейн
◆ 15 августа 1198
◆ Локация: ...
◆ Рядом: ...
◆ Состояние Акиры: ...
◆ Текущий фокус: ...
```

## Правила для любой шапки

- Шапка не должна быть длиннее самой сцены.
- Не писать в шапке скрытый лор.
- Не писать в шапке то, чего Акира не знает.
- Не писать будущие события календаря как уже произошедшие.
- Если день 15 августа завершён, не подтягивать шапку/план 15 августа заново.
