# Location Registry

Реестр **сыгранных, уточнённых и изменённых** локаций Академии Астрейн и предыстории.

Базовая карта хранится в `canon/academy_locations.md`.  
Этот файл не должен дублировать весь canon. Здесь фиксируются только места, которые уже важны для сцены, маршрута, памяти или будущего продолжения.

## Правило

Если локация появляется один раз как фон — можно оставить её в заметках сцены.

Если локация:

- повторяется;
- влияет на сюжет;
- связана с важной встречей;
- становится местом конфликта;
- содержит важные предметы;
- связана с тайной;
- меняет отношения персонажей;

её нужно закрепить здесь.

## Активные закреплённые локации

### `academy_back_court_exit`

- **Название в тексте:** задний выход Академии со стороны корта / баскетбольные площадки
- **Тип:** двор / спортивная зона / вход на территорию
- **Где находится:** боковая спортивная сторона Академии, рядом с кортом и баскетбольными площадками
- **Кто имеет доступ:** студенты, новички, персонал, охрана; это не закрытая зона
- **Как выглядит:** менее официальный вход, боковые дорожки, площадка, скамейки, разметка, стук мяча, меньше толпы, вид на тренировочные корпуса
- **Что там обычно есть:** студенты, играющие в баскетбол, наблюдатели у площадки, спортивные сумки, мячи, шум неофициального общения
- **Что там запрещено:** опасная энергия без инструктора, драки вне спарринга, порча имущества, оружие без допуска
- **Кто часто бывает:** Хару, спортивные студенты, потенциально Райден рядом с Хару
- **Сюжетное значение:** стартовая точка 15 августа 1198; первый живой социальный узел вместо сухого главного двора; место первого возможного пересечения с Хару
- **Скрытые детали:** маршрут выбран из-за меньшей толпы по совету Рэя; это не секретный тест куратора
- **Что уже произошло здесь:** стартовый state 15 августа поставлен сюда; Акира и Ливия приходят вместе

### `room_214`

- **Название в тексте:** комната 214
- **Тип:** комната общежития
- **Где находится:** общежитие новичков
- **Кто имеет доступ:** Акира, Ливия, комендант/куратор при необходимости; чужие гости только по сценовой логике
- **Как выглядит:** комната для заселения, личные места, окно, базовая мебель
- **Что там обычно есть:** багаж, выданная форма после регистрации, личные вещи после распаковки
- **Что там запрещено:** драки, опасное применение энергии, самовольные гости поздно ночью, ломать окна/замки
- **Кто часто бывает:** Акира, Ливия; Кир может появиться позже только по сценовой причине
- **Сюжетное значение:** первая безопасная бытовая точка Акиры и Ливии; место распаковки, разговоров и проверки окон
- **Скрытые детали:** окна комнаты 214 не заперты и не заблокированы
- **Что уже произошло здесь:** запланировано заселение 15 августа; переодевание в форму не считается случившимся без действия игрока

## Правило добавления

Новые локации добавлять только после появления в сцене или отдельного решения пользователя.

Не добавлять имена персонажей из старых сессий, тестовых новелл или другой игры.
