# NPC Registry: Академия Астрейн

Короткий реестр заметных NPC.

Здесь не должно быть случайно придуманных персонажей без причины. ИИ может предлагать NPC в сценах, но каноном NPC становится только после фиксации в этом реестре или в отдельной карточке.

## Правило имён

В тексте сцены имена пишутся русскими буквами.

Сами имена должны звучать японско-европейски, не по-русски.

Примеры подходящего стиля:

- Райден Стэрлинг;
- Хару Фостер;
- Ливия Кросс;
- Эйрен Вейл;
- Нора Эшфорд;
- Сайлас Вейн;
- Мина Роуэн.

Технические ID всегда пишутся латиницей в lowercase snake_case:

```txt
raiden_sterling
haru_foster
livia_cross
eiren_vale
```

## Когда NPC можно придумать

ИИ может придумывать NPC, если они нужны для живой академии:

- проверка новичков;
- общежитие;
- столовая;
- спортзал;
- стрельбище;
- тренировка;
- конфликт;
- слухи;
- поклонники;
- соперники;
- преподаватели;
- кураторы;
- медики;
- комиссия.

Но не каждый NPC сразу становится важным.

## Уровни NPC

### Фоновый NPC

Без имени или с кратким описанием.

Пример:

```txt
девушка у окна
старший курсант у входа
инструктор у стойки
```

Не добавлять в реестр, если он не влияет на сюжет.

### Названный NPC

NPC получил имя, но пока не важен.

Можно добавить коротко в сценовые заметки. В реестр добавлять только если он может вернуться.

### Повторяющийся NPC

NPC появился повторно, вступил в конфликт, флирт, слухи или связь с главным персонажем.

Добавить в этот файл с короткой анкетой.

### Ключевой NPC

NPC начал влиять на сюжет, отношения, репутацию, силу, тайну или будущие события.

Вынести в отдельный файл:

```txt
characters/npc/<id>.md
```

## Правила фиксации

- Не создавать NPC без причины.
- Не делать всех врагами.
- Не делать всех друзьями.
- Не делать всех романтическими интересами.
- Не делать всех одинаково полезными.
- Не давать NPC скрытый лор без причины.
- Если NPC стал важным — закрепить ID и больше не менять.
- Если NPC знает что-то, нужно указать, откуда он это знает.
- Если NPC отсутствовал в сцене, он не знает события этой сцены автоматически.
